1.全局安装 npm install -g create-react-app 2.切换到想要创建项目的目录，使用命令create-react-app react_staging（create-react-app + 你想创建应用的名字） 3.进入项目文件夹cd react_staging 4.启动项目： npm start public-静态资源文件 favicon--网站页签图标 index.html--主页面 manifest.json--应用加壳的配置文件 robots--爬虫协议文件 src-源码文件夹 App.css--App组件的样式 App.js--App组件 App.test.js--用于给App做测试 index.css--样式 index.js--入口文件 reportWebVitals.js--页面性能分析文件（需要web-vitals库的支持）
